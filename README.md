# VidFetch

A modern GUI for yt-dlp, built with Electron and Vue 3.

## Project Setup

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Project Structure

- `src/main`: Electron main process
- `src/preload`: Electron preload scripts
- `src/renderer`: Vue 3 renderer process
